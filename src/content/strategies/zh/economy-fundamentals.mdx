## 心态切换

OpenFront 的赢家不是占地最多的人，**而是经济曲线最陡的人**。如果你的开局是"打 bot → 占地 → 再打 bot"，AI 正在闷声攒金币，5 分钟后就有原子弹砸到你首都。

把目标从*扩张*换成*基建*。本页接下来是建造顺序、数学公式和踩坑清单。

## 为什么"把对手压在 20% 血量"其实是亏的

正面防御在没有建筑加成的情况下是净亏损：

- 地形 `mag` 仅 80 / 100 / 120（平原 / 高地 / 山地）
- **防御工事（Defense Post）**在 30 tiles 范围内提供 `mag ×5`、`speed ×3`
- 边境没有 Defense Post 时，你付 18% 换 20%，双方都在掉血——AI 回血更快，因为它没付损失的代价

5 万金币的 Defense Post 是**全游戏性价比最高的建筑**。前线先放 2-3 个再说其他。

## AI 的核弹时钟

为什么你刚占下一座 Factory，别人"已经"在丢核弹：

```
基础金币：    玩家 100/tick, Bot 50/tick, Easy Nation × 0.5
Missile Silo: 1,000,000 金币 (建造 100 ticks)
原子弹：      750,000 金币 (固定)
```

Easy AI 大约 5-7 分钟就能凑出 1.75M 金币。**5 分钟是威慑生死线**——错过了，你 SAM 还没动工就先吃核弹。

## 标准建造顺序（前 5 分钟）

| 时间 | 动作 | 成本 | 原因 |
|---|---|---|---|
| 0:00-1:00 | 干掉一个 small bot，冲到海岸 | — | 地块越多，人口上限越高 |
| 1:00-2:00 | 边境 2-3 个 Defense Post + 1 个 Port + 1 个 Factory | 150k + 125k + 125k | 锁前线，开两条收入流 |
| 2:00-4:00 | City / Port / Factory 升 2 级 | 各 250k | 每级翻倍产出 |
| 4:00-5:00 | 1 个 Missile Silo + 1 个 SAM | 1M + 1.5M | 哪怕没有原子弹，**SAM 本身就是威慑**——bot 核弹会绕开 70 tile 的覆盖圆 |

## 为什么你的贸易船被团灭

三件事叠加：

1. **贸易船在 Port 之间往返**。只有 1 个 Port 时，它会去*其他玩家*的 Port，包括没跟你开战的 AI。
2. **路径穿过敌方海域 = 必死**。v24 起 **战舰优先打击 Transport / Trade ship**，命中后立刻冷却——只要你和目标 Port 之间有一艘敌方战舰，整条贸易线就废了。
3. **内陆 Port 亏钱**。藏在领土深处的 Port 让贸易船必须绕远路、经过更多争议海域。

```
贸易船单趟收益：
floor( (75,000 / (1 + e^(-0.03 × (dist - 300))) + 50 × dist) × goldMultiplier )
```

距离 < 300 只拿到保底（约 50 × dist）。远距离回报高，但死得也快——这就是 tradeoff。

## 火车经济：更安全的钱

v24 起**单人和私密大厅有 Train**。Factory 自动生成，沿铁路在你的 City 和 Factory 之间运行，**战舰碰不到它**。

```
火车单次到站收益：
ally = 35,000  ·  team = 25,000  ·  self = 10,000
− 5,000 × max(0, citiesVisited − 9)
下限 5,000

生成速度： (factories + 10) × 15 ticks
```

自家路线一趟只有 10k，但你 100% 掌控这条线。**3 个 Factory 串 5 个 City 形成环路 = 敌方拦不到的稳定收入**。

## 优先级速查表

| 优先级 | 购买 | 成本 | 原因 |
|---|---|---|---|
| ★★★ | 前线 3-4 个 Defense Post | 各 50k | mag ×5，正面防御唯一不亏的方法 |
| ★★★ | 核心区 1 个 Port + 1 个 Factory | 各 125k（共享 n） | 两条平行收入流 |
| ★★ | Port / Factory / City 升到 L2-L3 | 250k → 500k → 1M | 每级翻倍产出 |
| ★★ | 5:00 之前 1 个 Missile Silo + 1 个 SAM | 1M + 1.5M | Bot 核弹绕开 SAM——买保险 |
| ★ | 多 Factory 跑火车环路 | 单座最多 1M | 收入免疫战舰 |

## 经典误区

- **没造 Factory 就盖第二个 Port**。Port 成本按 `n = ports + factories` 累计，2 个 Port 跟 1 Port + 1 Factory 同价——但 Factory 给你 Train，沉不了。
- **海岸尖角放裸 Port**。看起来贸易距离漂亮，可第一艘路过的战舰就把它砸了，125k 打水漂。海边 Port 旁边必须配一个 Defense Post。
- **等"金币够了"再造 SAM**。SAM 是你**安全攒金币的前提**。Bot 核弹专砸没防护的核心地块。
- **囤原子弹**。750k 一枚，一座 3 级 SAM 全拦。真正的核打击是氢弹（5M）。

## 决策树

```
你的边境有 ≥ 2 个 Defense Post 吗？
├─ 否 → 先造。其他都不重要。
└─ 是 → Port 和 Factory 都有吗？
    ├─ 否 → 把缺的那个补上 (125k)
    └─ 是 → 时间过 4:00 了吗？
        ├─ 否 → 建筑升到 L2
        └─ 是 → SAM 造了吗？
            ├─ 否 → 立刻 Silo + SAM，哪怕没原子弹
            └─ 是 → 攒到 5M 金币 → 氢弹
```

## 一句话总结

**5 分钟前你玩的是经济游戏，5 分钟后你玩的是核游戏。跳过第一阶段，你就没有第二阶段。**
