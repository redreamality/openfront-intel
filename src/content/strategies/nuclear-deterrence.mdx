## 三阶段核战节奏

OpenFront 的核战可以拆成三个阶段：

### 阶段 1 · 威慑建立（3-5 分钟）

- **目标**：让对手知道你能反击。
- **建造**：1 座 Missile Silo（1M）+ 1 座 SAM（1.5M）。
- **作用**：单座 SAM 射程 70 tiles，已能覆盖你的核心城市。AI 核弹会主动避开 SAM 覆盖区，意味着<strong>仅仅造出 SAM 就有威慑</strong>。

### 阶段 2 · 实弹打击（5-8 分钟）

- **目标**：积 5M 金币 + 1 枚氢弹砸对方核心。
- **建造**：2-3 座 SAM 升级到 3 级以上、第 2 座 Silo。
- **打击优先级**：
  1. 对方 City 群（消耗其部队上限）。
  2. 对方 Silo 与 SAM（瘫痪反击能力）。
  3. 对方贸易港（断经济）。

### 阶段 3 · MIRV 协同（8 分钟以上）

- **目标**：用 MIRV 强迫对方分散防御。
- **建造**：积 25M 金币，发射第 1 枚 MIRV；之后每发 + 15M。
- **要点**：v24 削弱了 MIRV 杀伤，<strong>价值在心理压力</strong>——SAM 拦不住 MIRV，迫使对手把贵 SAM 散开布点，反而让单点防御变弱。

## SAM 布点公式

每座 SAM 覆盖一个圆盘：

```
samRange(level) = 150 − 480 / (level + 5)
level 1: 70  · level 2: ~81 · level 3: ~90
level 4: ~96 · level 5: ~140
```

**两座以上覆盖同一个目标**才能扛齐射（冷却 90 ticks）。但不要堆在一起——一颗氢弹外环 100 tiles 能把它们全炸了。

## 反核 checklist

- [ ] 至少 2 座 SAM 覆盖每个核心 City
- [ ] SAM 间距 > 80 tiles
- [ ] Silo 紧贴 SAM 内侧
- [ ] City 升 2 级以上（人口缓冲）
- [ ] 不依赖单点防御——多个核心备份

## 经典误区

- **囤一堆 Atom Bomb 当主力** — 750k 一枚单次伤害有限，被一座 3 级 SAM 全拦。氢弹（5M）是真核打击。
- **MIRV 当大杀器** — v24 后 MIRV 弹头 outer 仅 18 tile，伤害远不如氢弹，把它当心理战工具。
- **裸 Silo 无 SAM** — 你的 Silo 会被对方氢弹优先拆掉，反击能力立即归零。

## 决策树

```
你有 5M 金币吗？
├─ 否 → 继续刷经济（City + Port + 升级），不要造核
└─ 是 → 对方有 SAM 吗？
    ├─ 否 → 直接氢弹拆 City 群
    └─ 是 → 对方 SAM 数 < 3？
        ├─ 是 → 齐射 2-3 枚原子弹覆盖 SAM 区
        └─ 否 → 积金币到 25M 上 MIRV
```
