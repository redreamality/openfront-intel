> v24 a introduit les raccourcis d'attaque (Engla) et de navires — les quelques touches les plus utilisées méritent d'être mémorisées par cœur.

## Clavier

| Touche | Action |
| --- | --- |
| Clic droit (votre territoire) | Ouvrir le menu Radial |
| Clic droit (ennemi / sans propriétaire) | Lancer directement une attaque ou un transport |
| Glisser le curseur Attack Ratio | Ajuster la part de Troops envoyée par attaque |
| `A` | Mode attaque (en direction du curseur) |
| `B` | Ouvrir le menu Build |
| `T` | Ouvrir l'interface de navire de transport |
| `Esc` | Annuler le menu en cours |
| `Space` | Pause (solo / salon privé) |

> Les touches réelles correspondent à ce qui est affiché dans Settings → Keybindings en jeu ; depuis la v24, certaines touches sont personnalisables.

## Menu Radial (refondu en v24)

Déclenché par un clic droit sur vos terres, sa disposition est la suivante :

```
       Build City
            |
    Port -- ⊕ -- Defense
            |
       Missile Silo
```

La v24 l'a transformé en menu à plusieurs niveaux : au-delà des six bâtiments principaux, le sous-menu donne accès aux options SAM, Factory et aux améliorations.

## Combinaisons d'actions

- **Salve nucléaire rapide** : `B → Atom Bomb → Click Target`, puis cliquez sur chaque cible ennemie l'une après l'autre.
- **Défense côtière** : `Build → Port → Build → DefensePost`, en collant le DefensePost contre l'intérieur du Port.
- **Alliance** : cliquez sur l'avatar de l'adversaire → `Request Alliance`.

## Vue et navigation

| Touche | Action |
| --- | --- |
| Molette de la souris | Zoom |
| Clic milieu maintenu | Déplacer la vue |
| `+ / -` | Zoom (alternatif) |
| Double-clic sur un avatar de joueur | Verrouiller la caméra sur son cœur |

## Rythme de combat

Les joueurs aguerris font tourner leur souris entre trois zones :

1. **Le centre de votre territoire** — construction et amélioration.
2. **La frontière** — attaques et transports.
3. **La minimap / la liste des joueurs** — pour suivre la courbe d'expansion des adversaires.

Quel que soit votre style de jeu, la règle d'or est <strong>de ne pas quitter la vue du champ de bataille</strong> — OpenFront est rapide, et 10 secondes d'absence peuvent suffire à perdre une City.
