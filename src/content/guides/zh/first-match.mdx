## 进入游戏前

打开 [openfront.io](https://openfront.io/)，建议先建一个账户（保留战绩与皮肤），再点 **Single Player** 开第一局。

第一局推荐设置：

- **地图**：Europe Classic（地理熟悉，瓶颈多）。
- **难度**：把 Nation 全部设为 Medium，Bot 数量 5。
- **模式**：FFA。
- **开关**：勾选 `Peace Time`，关闭核武器（取消勾 `Nukes`）。

## 进场第一分钟

1. **选址**：点地图上的国旗，建议法兰西半岛、意大利、巴尔干这类有海有瓶颈的位置。
2. **拉攻击条**：屏幕底部有 `Attack Ratio` 滑块，调到 60-70%。
3. **出生免疫**：开局头 5 秒任何攻击不会对你生效——放心扩张。
4. **首次扩张**：右键陆地相邻的 tile，把部队推到无主土地。

## UI 速读

| 位置 | 数字 | 含义 |
| --- | --- | --- |
| 左上 | Troops / MaxTroops | 部队当前 / 上限 |
| 左上 | Gold | 金币总数 |
| 屏幕下 | Attack Ratio | 每次攻击投入比例 |
| 右键空地 | Build Menu | 建造选择 |
| 右键自己土地 | Radial Menu | 快速建造、升级 |

## 第一座建筑：City

打开 Build Menu → 选 City（125,000 金币）。放在你领土中心位置，远离边境。建成后部队上限 +250,000。

> 没有 125k 金币？继续扩张 30-60 秒，基础金币 100/tick 会让你积累足够预算。

## 第一座 Port

City 建完后，开始攒第 2 笔 125k 投港口。港口必须放在海岸 tile（蓝色海与你陆地接壤的边缘）。建好后贸易船开始随机生成。

## 第一座防御工事

观察对手扩张方向，把 1 座 DefensePost（50k）放在<strong>对手最有可能进攻的边界</strong>，挡住对方一段时间足以让你撤退/反击。

## 半小时复盘

第一局不要心疼输赢。打完后看：

- 你扩了多大领土？最高 Tiles 数。
- 你和谁结过盟？盟约怎么破裂的？
- 对手什么节点开始造核？SAM 起了几座？

带着这些观察读 [游戏机制](/openfront-intel/mechanics/) 系列，下一局战绩会显著提升。
