> v24 hat Angriffs-Hotkeys (Engla) und Schiffs-Hotkeys eingeführt — die wenigen am häufigsten genutzten Tasten lohnen sich, ins Muskelgedächtnis zu übernehmen.

## Tastatur

| Taste | Aktion |
| --- | --- |
| Rechtsklick (eigenes Gebiet) | Radial-Menü öffnen |
| Rechtsklick (Gegner / herrenlos) | Direkt einen Angriff oder Transport starten |
| Attack-Ratio-Regler ziehen | Anteil der pro Angriff eingesetzten Troops anpassen |
| `A` | Angriffsmodus (in Richtung des Cursors) |
| `B` | Build-Menü öffnen |
| `T` | Transportschiff-Oberfläche öffnen |
| `Esc` | Aktuelles Menü abbrechen |
| `Space` | Pause (Solo / private Lobby) |

> Die tatsächlichen Belegungen richten sich nach Settings → Keybindings im Spiel; seit v24 sind einige Tasten anpassbar.

## Radial-Menü (in v24 überarbeitet)

Wird durch Rechtsklick auf eigenes Land geöffnet, der Aufbau:

```
       Build City
            |
    Port -- ⊕ -- Defense
            |
       Missile Silo
```

v24 hat es zu einem mehrstufigen Menü gemacht: neben den sechs Kerngebäuden zeigt das Untermenü SAM, Factory und Upgrade-Optionen.

## Aktions-Kombinationen

- **Schnelle Atomsalve**: `B → Atom Bomb → Click Target`, anschließend jedes feindliche Ziel der Reihe nach anklicken.
- **Küstenverteidigung**: `Build → Port → Build → DefensePost`, den DefensePost dicht an die Innenseite des Ports setzen.
- **Allianz**: auf das Avatar des Gegners klicken → `Request Alliance`.

## Sicht und Navigation

| Taste | Aktion |
| --- | --- |
| Mausrad | Zoomen |
| Mittlere Maustaste ziehen | Ansicht verschieben |
| `+ / -` | Zoomen (alternativ) |
| Doppelklick auf einen Spieler-Avatar | Kamera auf dessen Kern fixieren |

## Kampfrhythmus

Bei erfahrenen Spielern wechselt die Maus zwischen drei Bereichen:

1. **Die Mitte des eigenen Landes** — Bauen und Aufrüsten.
2. **Die Grenze** — Angriffe und Transporte.
3. **Die Minikarte / Spielerliste** — die Expansionskurve der Gegner beobachten.

Egal welcher Spielstil — die Kernregel lautet <strong>die Sicht auf das Schlachtfeld nicht zu verlassen</strong> — OpenFront ist schnell, und 10 Sekunden Abwesenheit können reichen, um eine City zu verlieren.
