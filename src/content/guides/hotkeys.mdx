> v24 引入了攻击热键（Engla）与船只热键，使用频率最高的几个键值得肌肉记忆。

## 键盘类

| 键位 | 行为 |
| --- | --- |
| 鼠标右键（自己领土） | 召出 Radial 菜单 |
| 鼠标右键（敌方/无主） | 直接发起攻击或运输 |
| 滑动 Attack Ratio 滑块 | 调整每次攻击投入比例 |
| `A` | 攻击模式（朝鼠标方向）|
| `B` | 召出 Build 菜单 |
| `T` | 召出运输船界面 |
| `Esc` | 取消当前菜单 |
| `Space` | 暂停（单人 / 私密大厅）|

> 实际键位以游戏内 Settings → Keybindings 为准；v24 起部分键位可自定义。

## Radial 菜单（v24 重做）

右键自己土地后召出，结构：

```
       Build City
            |
    Port -- ⊕ -- Defense
            |
       Missile Silo
```

v24 改为多级菜单：核心 6 个建筑外，二级菜单可见 SAM、Factory 与升级选项。

## 操作组合

- **快速齐射核弹**：`B → Atom Bomb → Click Target`，依次点对方目标。
- **海岸防御**：`Build → Port → Build → DefensePost`，把 DefensePost 紧贴 Port 内侧。
- **同盟**：点对方头像 → `Request Alliance`。

## 视野与导航

| 键位 | 行为 |
| --- | --- |
| 滚轮 | 缩放 |
| 中键拖动 | 平移视野 |
| `+ / -` | 缩放（备用）|
| 双击玩家头像 | 锁定视角到对方核心 |

## 实战节奏

熟练玩家的鼠标动作大多在三处轮转：

1. **自己土地的中部** — 建造与升级。
2. **边境** — 攻击与运输。
3. **小地图 / 玩家列表** — 看对手扩张曲线。

无论操作风格如何，核心是<strong>不要离开战场视野</strong>——OpenFront 节奏快，离开 10 秒可能就是一座 City 易主。
