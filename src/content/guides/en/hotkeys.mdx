> v24 introduced attack hotkeys (Engla) and ship hotkeys — the few most-used keys are worth committing to muscle memory.

## Keyboard

| Key | Action |
| --- | --- |
| Right click (your own territory) | Open the Radial menu |
| Right click (enemy / unowned) | Launch an attack or transport directly |
| Drag the Attack Ratio slider | Adjust the share of Troops sent per attack |
| `A` | Attack mode (towards the cursor) |
| `B` | Open the Build menu |
| `T` | Open the transport ship interface |
| `Esc` | Cancel the current menu |
| `Space` | Pause (single-player / private lobby) |

> The actual bindings follow what is shown in Settings → Keybindings in-game; since v24 some keys can be customised.

## Radial menu (reworked in v24)

Triggered by right-clicking your own land, the layout is:

```
       Build City
            |
    Port -- ⊕ -- Defense
            |
       Missile Silo
```

v24 turned it into a multi-level menu: beyond the six core buildings, the secondary menu exposes SAM, Factory, and upgrade options.

## Action combos

- **Quick nuke salvo**: `B → Atom Bomb → Click Target`, then click each enemy target in sequence.
- **Coastal defense**: `Build → Port → Build → DefensePost`, placing the DefensePost tight against the inside of the Port.
- **Alliance**: click the opponent's avatar → `Request Alliance`.

## View & navigation

| Key | Action |
| --- | --- |
| Mouse wheel | Zoom |
| Middle-click drag | Pan the view |
| `+ / -` | Zoom (fallback) |
| Double-click a player avatar | Lock the camera onto their core |

## Combat rhythm

Skilled players' mouse work rotates through three areas:

1. **The middle of your own land** — building and upgrading.
2. **The border** — attacks and transports.
3. **The minimap / player list** — watching rivals' expansion curves.

Whatever your play style, the core rule is <strong>don't leave the battlefield view</strong> — OpenFront moves fast, and 10 seconds away can be enough for a City to change hands.
